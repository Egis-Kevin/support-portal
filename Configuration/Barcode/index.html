Barcode Recognition Setup

Setup an extractFromBarcode rule on at he appropriate node Install the Softek SDK** Download the latest version from:
Windows SDK v8.1.2
Linux SDK v8.1.2

Libraries need to be copied to /usr/lib/java, /usr/lib/ or C:\Windows\System32 or to where the java.library.path is pointing\

For a list of all versions goto http://www.bardecode.com/en1/quick-download/ although a different license may need to be loaded via the License property
